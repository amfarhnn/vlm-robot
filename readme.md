# VLM Robot

`VLM Robot` is a research project focused on socially aware mobile robot navigation using vision-language models.

This project is centered on one primary paper:
- `VLM-Social-Nav: Socially Aware Robot Navigation through Scoring (2025)`

## Project Focus

The goal of this project is to study how a vision-language model can help a robot navigate in a way that is not only collision-free, but also socially appropriate around people and shared spaces.

The main research themes are:
- socially aware robot navigation
- vision-language reasoning for scene understanding
- scoring-based trajectory selection
- human-centered navigation behavior

## Research Direction

This project will use `VLM-Social-Nav` as the main reference for understanding:
- how social cues in the environment can be turned into navigation costs
- how VLMs can support higher-level navigation decisions
- how robot behavior can be improved beyond simple obstacle avoidance
- how socially compliant navigation can be evaluated in practice

## Project Goals

Possible goals for this repository include:
- summarizing the paper's method and architecture
- reproducing key ideas in a simplified prototype
- documenting datasets, assumptions, and experiment design
- comparing social navigation behavior against baseline navigation methods
